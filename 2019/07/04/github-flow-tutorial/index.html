Github flowд№Ӣд»Ӣзҙ№иҲҮеҜҰдҪң

Posted on 2019-07-04 Edited on 2019-12-15

Github flowжҳҜд»Җйәј?

Github flowжҳҜдёҖеҖӢд»ҘеҲҶж”ҜзӮәдё»и»ёзҡ„иј•йҮҸеҢ–е·ҘдҪңжөҒзЁӢгҖӮ

зӣёијғж–јGit flowеҲҶжҲҗдә”еӨ§еҲҶж”Ҝ(masterгҖҒhotfixгҖҒreleaseгҖҒdevelopгҖҒfeature)пјҢGithub flowеҸҜиӘӘжҳҜзӣёз•¶з°Ўе–®пјҢе…¶дё»и»ёеҸӘйңҖиҰҒеҲҶзӮәmasterи·ҹbranchеҚіеҸҜгҖӮ

з°Ўе–®дҫҶиӘӘе°ұжҳҜпјҢmasterдёҠйқўзҡ„жҜҸеҖӢзүҲжң¬йғҪжҳҜеҸҜз”Ёзҡ„пјҢиҖҢеҠҹиғҪзҡ„й–ӢзҷјеүҮз”ұеҲҶж”ҜдҫҶйҖІиЎҢпјҢз•¶еҠҹиғҪеҲҶж”Ҝй–ӢзҷјеҘҪпјҢдёҰдё”з¶“йҒҺй©—иӯүеҫҢпјҢеҶҚе°ҮжӯӨеҲҶж”ҜиҲҮmasterйҖІиЎҢеҗҲдҪөгҖӮ

Github flowзҡ„жөҒзЁӢпјҢеӨ§иҮҙеҸҜеҲҶзӮәд»ҘдёӢе…ӯйҡҺж®ө:

  1. ж–°еўһеҲҶж”Ҝ(Create a branch)
  2. жҸҗдәӨcommit(Add commits)
  3. жҸҗеҮәpull request(Open a Pull Request)
  4. code review(Discuss and review your code)
  5. йғЁеұ¬й©—иӯү(Deploy)
  6. еҗҲдҪөеҲҶж”Ҝ(Merge)

еҜҰдҪң

й–Ӣзҷјж–°еҠҹиғҪзҡ„жӯҘй©ҹеӨ§иҮҙеҰӮдёӢ:

  1. forkдёҖд»ҪеҺҹжң¬зҡ„жӘ”жЎҲеҲ°иҮӘе·ұзҡ„еёіиҷҹдёӢ
  2. е°Үforkзҡ„жӘ”жЎҲдёӢијүеҲ°йӣ»и…ҰдёҠ
  3. й–Ӣе•ҹж–°еҲҶж”ҜпјҢдёҰйҖІиЎҢй–Ӣзҷј
  4. е°Үй–ӢзҷјеҘҪзҡ„еҲҶж”ҜpushеҲ°иҮӘе·ұforkзҡ„repositoryдёҠ
  5. жҸҗдәӨpull request

1 ~ 2жӯҘй©ҹеҸӘжңүеңЁз¬¬дёҖж¬ЎжҷӮйңҖиҰҒйҖІиЎҢпјҢд»ҘеҫҢй–Ӣзҷјж–°еҲҶжһқеҸӘйңҖиҰҒйҮҚиӨҮжӯҘй©ҹ3 ~ 5еҚіеҸҜгҖӮ

д»ҘдёӢжҲ‘е°ҮзӨәзҜ„дёҖж¬Ўе®Ңж•ҙзҡ„ж“ҚдҪңжөҒзЁӢпјҢдёҰжҗӯй…Қsource treeзҡ„жЁ№зӢҖең–йҖІиЎҢи¬ӣи§ЈгҖӮиҰҒжіЁж„Ҹзҡ„жҳҜжҲ‘еҖ‘дҪҝз”Ёзҡ„еёіиҷҹ(иҰҒйҖІиЎҢpull requestзҡ„еёіиҷҹ)зӮәriver40333пјҢеҺҹе§Ӣе°ҲжЎҲзҡ„жүҖжңүиҖ…зӮәcreek0810гҖӮ

1. forkжӘ”жЎҲеҲ°иҮӘе·ұеёіиҷҹдёӢ

forkзҡ„ж–№ејҸеҫҲз°Ўе–®пјҢжҲ‘еҖ‘еҸӘйңҖиҰҒйҖІеҲ°еҺҹе§Ӣе°ҲжЎҲзҡ„й ҒйқўпјҢжҢүдёӢеҸідёҠи§’зҡ„forkжҢүйҲ•пјҢдҫҝе®ҢжҲҗж“ҚдҪңгҖӮ

жҺҘи‘—пјҢжҲ‘еҖ‘еҸҜд»ҘеҫһиҮӘе·ұзҡ„repositoryиЈЎй»һйҖІеҺ»forkйҒҺдҫҶзҡ„е°ҲжЎҲпјҢеҸҜд»ҘзңӢеҲ°е°ҲжЎҲеүҚйқўеҜ«зҡ„жҳҜжҲ‘еҖ‘иҮӘе·ұзҡ„еҗҚеӯ—пјҢдёӢйқўжңғжҸҗзӨәи©Іе°ҲжЎҲеҺҹжң¬зҡ„жүҖжңүиҖ…гҖӮ

2. е°Үforkзҡ„е°ҲжЎҲдёӢијүеҲ°йӣ»и…ҰдёҠ

жіЁж„ҸпјҢиҰҒдёӢијүзҡ„жҳҜforkйҒҺдҫҶзҡ„е°ҲжЎҲпјҢиҖҢйқһеҺҹе§Ӣзҡ„е°ҲжЎҲгҖӮ
жҲ‘дҪҝз”Ёзҡ„еёіиҷҹжҳҜriver40333пјҢжүҖд»ҘдёӢијүзҡ„з¶ІеқҖзӮәhttps://github.com/river40333/github-flow-demo

еңЁдёӢијүеӣһдҫҶеҫҢпјҢжҲ‘еҖ‘еҸҜе·Іе°ҮеҺҹе§Ӣе°ҲжЎҲзҡ„йҒ з«ҜзүҲжң¬жҺ§еҲ¶еҖүеә«д№ҹеҠ е…Ҙиҝ‘дҫҶпјҢж–№дҫҝжҲ‘еҖ‘д№ӢеҫҢжӣҙж–°иҮӘе·ұforkеӣһдҫҶзҡ„masterеҲҶж”ҜгҖӮ

git remote add remoteName remoteURL
// ж–°еўһеҗҚзӮәremoteNameзҡ„йҒ з«ҜеҖүеә«пјҢе…¶з¶ІеқҖзӮәremoteURL

еҲ©з”Ёgit remote -vпјҢжҲ‘еҖ‘еҸҜд»ҘзңӢеҲ°еҺҹе§Ӣе°ҲжЎҲзҡ„йҒ з«ҜеҖүеә«е·Із¶“жҲҗеҠҹеҠ е…ҘгҖӮ

3. й–Ӣе•ҹж–°еҲҶж”ҜпјҢдёҰйҖІиЎҢй–Ӣзҷј

йҖҷиЈЎжҲ‘дҪҝз”Ёзҡ„жҢҮд»ӨжҳҜ

git checkout -b branchName
// ж–°е»әеҗҚзӮәbranchNameзҡ„еҲҶж”ҜпјҢдёҰеҲҮжҸӣиҮіжӯӨеҲҶж”Ҝ

4. е°Үй–ӢзҷјеҘҪзҡ„еҲҶж”ҜpushеҲ°иҮӘе·ұforkзҡ„repositoryдёҠ

йҖҷиЈЎеҒҮиЁӯжҲ‘еҖ‘е·Із¶“й–Ӣжі•еҘҪfeature1зҡ„еҠҹиғҪпјҢдёҰе·Із¶“йҖІиЎҢcommiteпјҢеҸҜд»ҘзңӢеҲ°жЁ№й•·йҖҷжЁЈгҖӮ

еңЁеҜҰйҡӣжҠҠеҲҶж”ҜpushеҲ°йҒ з«ҜеҖүеә«еүҚпјҢиЁҳеҫ—дёҖе®ҡиҰҒе…Ҳе°ҮиҮӘе·ұзҡ„masterеҲҶж”Ҝи·ҹеҺҹе§Ӣе°ҲжЎҲзҡ„masterеҗҢжӯҘпјҢдёҰдё”и·ҹжҲ‘еҖ‘й–Ӣзҷјзҡ„еҲҶж”ҜйҖІиЎҢеҗҲдҪөпјҢйҖҷжЁЈжүҚеҸҜд»Ҙеҫ—еҲ°жҜ”ијғд№ҫж·Ёзҡ„жЁ№гҖӮ

е°Үжң¬ж©ҹеҖүеә«зҡ„masterиҲҮеҺҹе§Ӣе°ҲжЎҲйҖІиЎҢеҗҢжӯҘпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢжҢҮд»Ө

git checkout master
// е°Үзӣ®еүҚжүҖеңЁзҡ„еҲҶж”ҜеҲҮжҸӣзӮәmaster
git pull вҖ“rebase upstream master
// е°ҮupstreamйҒ з«ҜеҖүеә«зҡ„masterеҲҶж”ҜжҠ“дёӢдҫҶпјҢдёҰиҲҮзӣ®еүҚжүҖеңЁзҡ„еҲҶж”Ҝд»Ҙreabseзҡ„ж–№ејҸйҖІиЎҢеҗҲдҪө


е°ҮmasterеҲҶж”ҜиҲҮеҺҹе§Ӣе°ҲжЎҲзҡ„masterйҖІиЎҢеҗҢжӯҘеҫҢпјҢеҸҜд»ҘзңӢеҲ°жҲ‘еҖ‘зҡ„masterеҫҖеүҚй•·дәҶпјҢдёҰдё”еҮәзҸҫfeature1еҲҶж”ҜгҖӮ

зӮәдәҶзўәдҝқд№ӢеҫҢpull requestеҗҲдҪөжҷӮзҡ„жЁ№й•·еҫ—д№ҫж·ЁпјҢжҲ‘еҖ‘йңҖиҰҒе°Үй–ӢзҷјеҘҪзҡ„еҲҶж”ҜиҲҮжӣҙж–°йҒҺеҫҢзҡ„masterеҲҶж”ҜйҖІиЎҢеҗҲдҪөпјҢжӯӨжҷӮжҲ‘еҖ‘еҸҜд»ҘдҪҝз”Ёд»ҘдёӢжҢҮд»Ө

git checkout feature1
// е°Үзӣ®еүҚжүҖеңЁзҡ„еҲҶж”ҜеҲҮжҸӣзӮәfeature1
git rebase master
дҪҝз”Ёreabaseзҡ„ж–№ејҸпјҢе°Үз•¶еүҚеҲҶж”ҜиҲҮmasterеҲҶж”ҜеҗҲдҪө


еҗҲдҪөе®ҢеҫҢпјҢеҸҜд»ҘзңӢеҲ°жҲ‘еҖ‘зҡ„featureеҲҶж”ҜиҲҮmasterеҲҶж”ҜеңЁеҗҢдёҖжўқз·ҡдёҠгҖӮ

жңҖеҫҢпјҢе°ҮжҲ‘еҖ‘иЈҪдҪңеҘҪзҡ„еҲҶж”ҜpushеҲ°иҮӘе·ұзҡ„йҒ з«ҜеҖүеә«пјҢе°ұеҸҜд»Ҙй–Ӣе§ӢйҖІиЎҢpull requestдәҶгҖӮ

5. жҸҗдәӨpull request

еӣһеҲ°жҲ‘еҖ‘forkзҡ„repositoryзҡ„й ҒйқўпјҢжә–еӮҷйҖІиЎҢpull requestгҖӮ

ијёе…Ҙзӣёй—ңиіҮиЁҠеҫҢпјҢжҢүдёӢеҸідёӢи§’зҡ„pull requestжҢүйҲ•пјҢеҚіе®ҢжҲҗж“ҚдҪңгҖӮ

е°ҮйҸЎй ӯиҪүеҲ°еҺҹе§Ӣе°ҲжЎҲзҡ„жҢҒжңүеёіиҷҹпјҢжҲ‘еҖ‘еҸҜд»ҘзңӢеҲ°жңүдёҖеҖӢж–°зҡ„pull requestгҖӮ

й»һйҖІжғізңӢзҡ„pull requestеҫҢпјҢеҸҜд»ҘзңӢеҲ°жҸҗеҮәpull requestжҷӮи©ІеҲҶж”ҜдҪңиҖ…з•ҷдёӢзҡ„иЁҠжҒҜпјҢеҰӮжһңиҰәеҫ—жӯӨеҲҶж”ҜokпјҢе°ұеҸҜд»ҘжҢүдёӢMerge pull requestйҖІиЎҢеҗҲдҪөдәҶпјҒ

Merge pull requestзҡ„йҒёй …пјҢжҲ‘йҖҡеёёжңғйҒёж“Үcreate a merge commitпјҢйҖҷжЁЈе°ұеҸҜд»ҘзңӢеҲ°жңүеҲҶж”Ҝзҡ„жјӮдә®жЁ№ең–дәҶгҖӮ

еҰӮжһңжңүжғіе°ҚжҸҗеҮәpull requestзҡ„дҪңиҖ…иӘӘпјҢд№ҹеҸҜд»ҘеңЁдёӢйқўзҡ„ең°ж–№з·ЁијҜпјҢйҖІиЎҢиЁҺи«–гҖӮ

жңҖеҫҢпјҢеҺҹе§Ӣе°ҲжЎҲзҡ„жЁ№й•·йҖҷжЁЈгҖӮ
